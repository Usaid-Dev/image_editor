# Image Editor App

A completely responsive Image Editor App- Works on Android & IOS

## Features
- Pick Image
- Add Text
- Delete Text
- Bold Text
- Italic Text
- Add New Line
- Align Text Left, Right & Center
- Change Colour
- Increase Font Size 
- Decrease Font Size 
- Save Image To Gallery
